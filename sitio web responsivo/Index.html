Programa Educativo

Propósito

El alumno implementará sistemas empleando tecnologías web para extender la disponibilidad y alcances de los servicios de la organización.

Cuatrimestre

Cuatrimestre: Séptimo
Total de horas: 75
Modalidad: Presencial

Unidad I. Introducción a los sistemas web

Temas de aprendizaje

1. Introducción a la programación web
2. Configuración de servidores de desarrollo
3. Configuración de servidor de producción

Unidad II. Programación del lado del cliente

Temas de aprendizaje

1. Maquetación (HTML)
2. Hojas de estilo (CSS)
3. JavaScript

Unidad III. Programación del lado del servidor

Temas de aprendizaje

1. Interfaz de programación de aplicaciones y programación de tres niveles
2. Acceso a servicios
3. Monitoreo y estadísticas del sistema web

Unidad IV. Autenticación de usuarios

Temas de aprendizaje

1. Autentificación e interfaces seguras
2. Manejo de sesiones
3. Manejo de excepciones
